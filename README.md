## 🌍 About Vishwa Leader
Vishwa Leader is an esteemed international platform dedicated to global advocacy, thought leadership, and academic discourse. Originally founded as a pioneering print magazine over a decade ago, it is now evolving into a digital-first global forum, culminating in significant events like the **Business Summit** and **SOAS 2026 Conference**.

Our mission is to foster a network of global visionaries, industry pioneers, and academic scholars to shape discussions on sustainable leadership, social equity, and economic progress.

## 🚀 Key Features

* **Digital Archives**: Complete repository of official circulars, academic abstracts, and a rich history of Vishwa Leader print magazines dating back over a decade.
* **Global Conferences**: Annual international conferences bringing together leaders from around the world.
* **Thought Leadership**: Deep dives and academic articles spanning diverse disciplines.
* **Exclusive Memberships**: Access premium resources, networking opportunities, and priority event registrations.

## 🛠️ Technology Stack
This platform is built with cutting-edge technologies to ensure a robust, accessible, and dynamic user experience:

* **Framework**: [Next.js](https://nextjs.org/) (React)
* **Styling**: [Tailwind CSS](https://tailwindcss.com/)
* **Database**: Firebase / Firestore
* **Deployment**: [Vercel](https://vercel.com)

## 📦 Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🌐 Deployment
This project is continuously deployed via Vercel. 
To deploy a production build:
```bash
npx vercel --prod
```
